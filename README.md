# ESP32-Dev-Kit
KiCad project files for an ESP32 development board — schematic, PCB layout, and footprints for a custom dev kit design.
ESP32 Dev Kit — KiCad Design Files
This repository contains the complete KiCad project for a custom ESP32 development board, including schematic capture, PCB layout, and associated footprints/symbols.
Contents:

Schematic (.kicad_sch) — ESP32 module wiring, power regulation, USB-to-serial interface, and GPIO breakout
PCB layout (.kicad_pcb) — routed board with defined net classes and design rules
Footprint and symbol libraries used in the design
Gerber/production files (if included — add or remove this line)

Features:

ESP32-WROOM/WROVER module support
USB-to-UART programming circuit
Onboard 3.3V voltage regulation
Breakout headers for GPIO access
Reset/boot button configuration

Tools used: KiCad 10.0
Feel free to open, modify, or fork this design for your own ESP32-based projects.
